Data Ingestion

EHR data streamed via FHIR APIs to local AI Zone. Patient identifiers removed at source.

2
Feature Engineering

Standardized clinical features extracted (vitals, labs, diagnoses, medications) using ML pipelines.

3
Local Training

Deep learning model trained on local data. Weights stored in encrypted format.

4
Privacy Application

Differential privacy noise added. Gradient clipping ensures bounded sensitivity.

5
Secure Transmission

Encrypted gradients sent to central server via mTLS. Zero knowledge of individual contributions.

6
Global Aggregation

FedAvg algorithm combines updates. New global model tested and distributed.
